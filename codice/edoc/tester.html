Testing module containing useful function for testing.

Authors: Massimiliano De Luise, Massimo Comuzzo.

Description

Testing module containing useful function for testing. The module contains test generator functions and simple test functions. The test generator functions are nedded for removing the default test timeout of eunit (5 seconds).
